Data Sources Discovery

Overview#

DataSource Discovery Engine (DSDE) is a tool that allows Digital Enabler users to find useful data all over the web. When user inserts a search, DSDE automatically browses web pages, extracts information out of them and ranks data according to their compliance to the user search.

Landing page#

After logging the user will be redirected to the tool landing page as you can see in the picture below.

In the top-right corner of the page, within then navbar, there is the user menu, where the user can access his platform reserved area or logout.

The main content of the page shows the list of searches started by the current user. Here both the completed and still running searchs are shown.

Start a new Search#

You can click on the ADD SEARCH button in order to acces the "New search" form.

As you can see in the picture below it is possible to set discovery criteria to find data sources over web.

In title field you can type name of your search. This doesn't affect the search results, it is only for user convenience. The Context field is intended to supply a keyword to limit the search boundaries. It can be the name of a state, city or region or whichever non-geographic scope you want to limit the search to. The Language dropdown field is where the user can choose the search language. It improves the quality of the results. By default it's set to English. The Keywords field is intended to contain a list of comma-separated words. This list will be used by the tool to evaluate the relevance of the scraped pages against the user interests. The Max Dept is the maximum dept the spiders can reach in web tree traversal.

After clicking on create button the user will be redirected to landing page where the just created search will be immediately available, but not yet started.

Start Searching#

There are two button on is for start and other is for delete search criteria.

But clicking on start button searching will start new button with name details will be available.

By clicking on details button new page will be open.

Stop Searching#

Also there are u button stop searching. By clicking on this button web crawler will stop to work. Once you stop the search you can't start it once again. As you can see on photos in drop-down details status will changed to stop.

Search result#

On photo you could see the results of search. Also you could use filter in as you can see on photo below. There are also a status for data sources in four level (Good , normal , bad , unranked). Also various filtering options. By default page result is not checked.

There are also filter for number or results that means what is the number of match that you want to see or export and Min Score ( >0.7 - is Good , >0.5 is Normal etc ).

There are option to export result (download as json file to you local machine) by clicking on export button in right part of page.

About stats#

If you check Pages you will see all web sites that are found. You can go to pages by clicking on the link. And also if some web domain have multiple pages you can go to each by clicking on page box below that sources. Also in the right last row you can see the score of sources. They are sorted from Very High to Low. On the left side there are a row that is called type. There are three possible types (File, Web and Portals)
