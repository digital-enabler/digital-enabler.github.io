Work area guide

The Mashup editor interface that you will find in the work area is composed of the following elements:

On the left you will find a side bar expandible by clicking on the arrow on the left of the screen. The side bar is composed of the following elements:

  1. Mashups: the list of all the mashups available in the work area. You can create a new mashup by clicking on the "CREATE MASHUP" button on the top right of the list or import it with the "IMPORT MASHUP" button. You can also search for a specific mashup by typing its name in the search bar on the top of the list.

  2. Datasources: the list of all the datasources available in the work area. You can create a new datasource by clicking on the "CREATE DATASOURCE" button on the top right of the list or import it with the "IMPORT DATASOURCE" button. You can also search for a specific datasource by typing its name in the search bar on the top of the list.

  3. Flows: the list of all the flows available in the work area. You can create a new flow by clicking on the "CREATE FLOW" button on the top right of the list or import it with the "IMPORT FLOW" button. You can also search for a specific flow by typing its name in the search bar on the top of the list.

  4. Account and settings

For each of these elements you can click on the three dots on the right of the name to open a menu with the following options:

  • Share: to share the element with some restriction:

    You can share the element via username and let them be able to edit the element. At the bottom of the menu you can find the label "Share with limitation". This option can be used, for example, to let visible your mashup for all the member of your organization.

  • Export: to export the element to share with alternative methods (e.g. email) and let the other users import with the functionality "IMPORT MASHUP/DATASOURCE/FLOW"

  • Duplicate: to duplicate the element

  • Delete: to delete the element

Mashup work area guide​

The Mashup editor interface that you will find in the work area is composed of the following elements:

  1. Toolbar
  2. Input node
  3. Output node
  4. Operator list
  5. Generic options:
    • Configure: rename, tags, choose the Smart Data Model producted by this mashup
    • Flow: to open the flow list where the mashup is used
    • Share
    • Duplicate
    • Delete
    • Export
    • Release: create a new release for this mashup and use it in a flow

To know how to use the mashup editor's operator we suggest to read the Mashup editor guide.

Operator's toolbar​

Each operator has a toolbar with the following elements:

  • Preview: to preview the result of the operator
  • Help: to open the help page of the operator
  • Delete: to delete the operator
  • Cache: to enable/disable the cache for the operator

Cache System​

The cache is generated based on the input/operator pair. This system associates the input with the operator as a single pair, allowing multiple values to be stored simultaneously. The main objective is to speed up the execution of a Mashup.

A clear example of this mechanism is the use of a token with a predetermined expiration. It's possible to define the token's validity period by entering the required time for its expiration into the "lease time".

The settings for the cache system are:

  • Status: to choose if enable or not the cache system
  • Lease time: Amount of time(ms) for which the value should remain cached.

Input and output node​

The input and output nodes are the nodes that you can use to connect the mashup to the outside world. The input node is the node that you can use to set the input values of the mashup. The output node is the node that you can use to get the output values of the mashup.

Input node settings​

1.Label: the label of the input node 2. Type: the type of the input node:

  • Query
  • Header
  • Body
  1. Content:
  • JSON: to set the input node content as a JSON
  • Text: to set the input node content as a text
  • Date: to set the input node content as the current date
  1. Default value: the default value of the input node

Toolbar​

The toolbar is composed of the following elements:

  1. Save: to save the mashup
  2. Settings: to set default values for the input nodes
  3. Undo: to undo the last action
  4. Redo: to redo the last action
  5. Zoom out: to zoom out the mashup
  6. Zoom in: to zoom in the mashup
  7. Preview: to preview the result of the mashup, with two different options:
    • Tree view: to preview the result of the mashup in a tree structure
    • Raw view: to preview the result of the mashup in a raw view
